Hack ISU

During Hack ISU, I used the previous knowledge of Android development to help our team developing a very practical app that can save you from getting kidnapped or get lost.

I was responsible for:

  • Designing the entire user interface for the app, as well as implementing most of them.

  • Help setting up background service for the app to run at the background.

I learned:

  • Basic JavaScript & Database knowledge
  • Using Twilio API to send text messages
  • How background service works in Android apps
